# ⚽ Fut-TV

A modern web application for tracking Football matches, standings, broadcasts, lineups, and live match information.
(Right now supporting only FIFA World Cup 2026)

## 🌍 Overview

Fut-TV was built to provide football fans with a clean and responsive experience for following the FIFA World Cup.

The application aggregates essential match information in a single place, including:

* 📺 Where to watch each match
* ⚽ Live scores
* 🏆 Group standings
* 📋 Team lineups
* 📈 Match events and timeline
* 📅 Upcoming fixtures
* 🎯 Match details and statistics

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/fut-tv.git
```

Navigate to the project folder:

```bash
cd fut-tv
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

## 🏗️ Build for Production

```bash
npm run build
```

Preview production build locally:

```bash
npm run preview
```
